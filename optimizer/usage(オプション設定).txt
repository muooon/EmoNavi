EmoNavi Usage 使い方

|||オプション指定方法||| tool側の引数設定に合わせてください

shadow オフ(False にする)：
use_shadow=False

動的学習率と感情スカラー等の現在値を取得(ツール側などから取得する)：
writer=writer

外部ツール(TensorBoard等)で値を把握したい場合は Optimizer 初期化時に SummaryWriter を渡してください
writer = SummaryWriter(log_dir="./runs/emonavi")
optimizer = EmoNavi(model.parameters(), lr=1e-4, writer=writer)
tensorboard --logdir=./runs/emonavi

|||Usage examples||| Please configure the arguments according to the tool's specifications.

Shadow off:
use_shadow=False

Monitor values with external tools (TensorBoard):
writer=writer

If you wish to monitor values using external tools (such as TensorBoard), please pass a SummaryWriter when initializing the Optimizer.
writer = SummaryWriter(log_dir="./runs/emonavi")
optimizer = EmoNavi(model.parameters(), lr=1e-4, writer=writer)
tensorboard --logdir=./runs/emonavi
